# Suivi analytique FEDER — Projet CAPs · Coll.in

Outil de suivi budgétaire du projet **CAPs : des communautés apprenantes au service de l'inclusion numérique**.

- **Référence dossier :** OCC015401
- **Cofinanceur :** FEDER — Région Occitanie
- **Période :** octobre 2025 – avril 2027

## Accès

L'outil est accessible à l'adresse :  
👉 `https://[votre-compte].github.io/caps-feder/`

## Données

Les données sont sauvegardées dans le navigateur (localStorage) et exportables en JSON.  
Le fichier de sauvegarde mensuel est déposé dans le dossier Nextcloud :  
`Coll.in / 1. Documents CA / Appels à projet - subventions / CAPs-FEDER / Suivi/`

## Usage mensuel

1. Ouvrir l'URL GitHub Pages
2. Cliquer **Import JSON** → charger la dernière sauvegarde depuis Nextcloud
3. Saisir les dépenses du mois
4. Cliquer **Export JSON** → déposer sur Nextcloud
5. Exporter le **Bilan narratif** pour le Bureau

## Accès restreint (optionnel)

Le dépôt peut être mis en **privé** sur GitHub — GitHub Pages reste accessible  
via URL pour les personnes qui ont le lien, sans authentification.
